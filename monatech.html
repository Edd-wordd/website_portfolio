Project

MonaTech LLC

In my role as a software engineer, I collaborated with a design and development company on a project aimed at bolstering the digital presence of local businesses and fostering server development for emerging entrepreneurs.

The tech stack I utilized comprised React JS, Material UI, Node JS, Express JS, Webpack, Babel, Jest, AWS Amplify, API Gateway, and Lambda. Each of these technologies played a pivotal role in the successful development of the website.

React JS and Material UI were the cornerstones of our front-end development, creating a dynamic and visually appealing user interface. Node JS and Express JS powered our back-end operations, ensuring smooth data flow. Webpack and Babel were essential for bundling JavaScript files and transpiling ES6 code into a version compatible with various browsers. Jest provided a comprehensive testing framework, ensuring code reliability. AWS Amplify, API Gateway, and Lambda streamlined our deployment process and managed serverless functions.

The culmination of these efforts was a user-friendly, functional website that effectively supported local businesses in marking their digital footprint and provided a launchpad for aspiring server developers.
