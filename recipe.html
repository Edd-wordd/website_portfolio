Project

Recipe App

I developed an application that enables users to search for and adjust recipes, as well as add their own. This was built using HTML, CSS, JavaScript, and Parcel.

The technologies I used for this project were simpler but no less effective. The core of the application was built with HTML, CSS, and JavaScript, with Parcel serving as the bundler.

HTML and CSS were used to create a user-friendly interface, while JavaScript powered the functionality, enabling users to search for, adjust, and add recipes. Parcel bundled the application, ensuring compatibility across different browsers.

The result of this work is a practical and user-friendly application that allows users to customize their culinary experiences by searching for, adjusting, and adding recipes.
